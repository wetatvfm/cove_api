== Introduction ==
A Drupal 7 module that implements the PBS Cove API.

== Installation ==
Install as any other module.

== Configuration ==
Grant permission "Administer Cove API settings" to user role who should edit
the API settings.

Next, go to admin/config/media/cove_api to set the API key and secret. If you
do not have a key and secret, see:
https://projects.pbs.org/confluence/display/coveapi/Welcome#Welcome-HowtoRequestaKey
